# Funzioni definite a tratti

Il dominio è diviso in parti, e per ogni parte il grafico è descritto da una
funzione diversa.

**Esempio**:

$$
f(x) = |x| = \begin{dcases}
x \geq 0 & \implies x  \\
x < 0    & \implies -x
\end{dcases}
$$

# Trasformazioni del grafico di una funzione

## Riflessione

- rispetto all'asse $x$: $f(x) \to -f(x)$;
- rispetto all'asse $y$: $f(x) \to f(-x)$;

### Traslazione

- verticale: $f(x) + a$:
  - se $a > 0$: il grafico si sposta verso l'alto;
  - se $a < 0$: il grafico si sposta verso il basso;
- orizzontale: $f(x + a)$:
  - se $a > 0$: il grafico si sposta verso sinistra;
  - se $a < 0$: il grafico si sposta verso destra;

### Riscalamento

- verticale: $k\ f(x)$:
  - se $k > 1$: il grafico si espande;
  - se $k < 1$: il grafico si comprime;
- orizzontale: $f(k\ x)$:
  - se $k > 1$: il grafico si comprime;
  - se $k < 1$: il grafico si espande;

## Inversa di una funzione

Data $f: [0, +\infty) \to \R$;
