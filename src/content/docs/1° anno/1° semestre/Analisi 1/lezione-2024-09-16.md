# Restrizione di una funzione

Data $f: A \to B$, la restrizione $f_{\vert D}$ definisce la funzione solo nel
sottoinsieme $D \subset A$.

**Esempio**:

- $f: \R \to \R, \ f(x) = \lfloor x \rfloor$: non è iniettiva.
- $D = \N, \ f_{\vert D}: \N \to \R, \ f(x) = \lfloor x \rfloor$: è iniettiva.

Di solito la restrizione si usa per fare in modo che una funzione acquisisca
certe proprietà in un determinato intervallo.

# Iniettività e suriettività dal punto di vista geometrico

- $f$ è iniettiva se una qualsiasi retta orizzontale tracciata nel piano
  cartesiano interseca $f(x)$ al massimo una volta.
- $f$ è suriettiva se una qualsiasi retta orizzontale tracciata nel piano
  cartesiano interseca $f(x)$ almeno una volta.
- $f$ è biettiva se una qualsiasi retta orizzontale tracciata nel piano
  cartesiano interseca $f(x)$ esattamente una volta.

# Funzioni limitate

- funzione limitata superiormente:
  $\exists\ M \in \R \mid \forall\ x \in \R, f(x) \leq M$;
- funzione limitata inferiormente:
  $\exists\ m \in \R \mid \forall\ x \in \R, f(x) \geq m$;
- funzione limitata: è una funzione limitata sia superiormente che
  inferiormente;

# Funzioni simmetriche

- funzione pari: $f(x) = f(-x)$;
- funzione dispari: $f(-x) = -f(x)$;
- funzione periodica: $\exists\ T \in \R, k \in \Z \mid f(x + k\ T) = f(x)$;
- funzione inversa nel piano cartesiano: $graf(f^{-1})$ è la simmetria di
  $graf(f)$ rispetto alla retta $y = x$.

# Funzioni monotone

- crescente:
  $x_{1}, x_{2} \in \R,\ x_{1} < x_{2} \implies f(x_{1}) \leq f(x_{2})$;
- strettamente crescente:
  $x_{1}, x_{2} \in \R,\ x_{1} < x_{2} \implies f(x_{1}) < f(x_{2})$;

Esistono anche le varianti decrescente e strettamente decrescente.
